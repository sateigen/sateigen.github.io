My Portfolio

eduFactor

I conceptualzied eduFactor to provide students with a safe learning environment to practice answering the various types of questions they will encounter on standardized assessments. By creating a step-by-step tutorial, students are instructed as to how they will need to interact with the questions they face and recieve feedback on their progress. This web application required implementing skills in Python, Django, JavaScript, HTML, and CSS.

Flix

As my first Django project, Flix taught me to apply my Python skills to learning the Django framework. I was also able to manipulate a large database through the use of PostgreSQL. The application allows users to view movies that have been rated, register, login, and logout. Once logged in, users can rate movies and get reccomendations of movies to watch.

View the github repository.

Road Rage

This was a pair-programming project aiming to determine the optimal speed limit for a 1km stretch of road. With a team, I utilized object-oriented programming and dove into the concept of simulations. Using matplotlib, we were able to graphically display the data that we recorded in a Jupyter Notebook.
